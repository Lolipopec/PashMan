# PashMan

Локальный клиент HTTP-запросов в одном исполняемом файле.
Запускается, поднимает локальный веб-сервер на `127.0.0.1` и открывает интерфейс
в браузере.

- Один бинарник под ОС, **без установки и без зависимостей** (только стандартная библиотека Go).
- Данные хранятся в обычных JSON-файлах в папке `collections` рядом с программой. Базы данных нет.
- Сервер слушает только `127.0.0.1` — наружу ничего не открывается.

## Возможности

- **Одиночный запрос**: методы GET/POST/PUT/PATCH/DELETE/HEAD/OPTIONS, заголовки, тело JSON.
- **Авторизация**: без неё, Basic (логин/пароль), Bearer-токен.
- **Редактор query-параметров** с двусторонней синхронизацией с полем URL.
- **Прогон по CSV**: подстановка `{{колонка}}` в URL, тело, заголовки и поля авторизации;
  любой разделитель; **редактирование CSV в табличном виде**; результаты в реальном времени;
  кнопка «Остановить»; пауза между запросами.
- **Фильтр и поиск по результатам** прогона (по статусу и по всем полям), **поиск с подсветкой по телу ответа**.
- **Коллекции**: сохранение (умное — обновление по месту без копий), импорт из JSON-файла,
  переименование/дублирование/удаление запросов, переименование коллекций.

## Запуск

Готовые бинарники — скачать напрямую:

| Скачать | ОС |
|---------|----|
| ⬇️ [**pashman-win.exe**](https://github.com/Lolipopec/PashMan/raw/main/dist/pashman-win.exe) | Windows 10/11 |
| ⬇️ [**pashman-mac-arm64**](https://github.com/Lolipopec/PashMan/raw/main/dist/pashman-mac-arm64) | macOS на Apple Silicon (M1–M4) |
| ⬇️ [**pashman-mac-intel**](https://github.com/Lolipopec/PashMan/raw/main/dist/pashman-mac-intel) | macOS на Intel |

> Ссылки `…/raw/main/…` отдают файл напрямую (скачивание начнётся сразу).

Подробная инструкция для пользователя (включая снятие карантина Gatekeeper на macOS) —
[`dist/ИНСТРУКЦИЯ.md`](dist/%D0%98%D0%9D%D0%A1%D0%A2%D0%A0%D0%A3%D0%9A%D0%A6%D0%98%D0%AF.md).

Кратко:
- **Windows** — двойной клик по `pashman-win.exe` (при предупреждении SmartScreen: «Подробнее» → «Выполнить в любом случае»).
- **macOS** — в Терминале: `chmod +x ./pashman-mac-arm64 && xattr -d com.apple.quarantine ./pashman-mac-arm64 && ./pashman-mac-arm64`.

Интерфейс откроется на `http://127.0.0.1:8787` (если порт занят — следующий свободный).

## Сборка из исходников

Нужен [Go](https://go.dev/dl/) 1.22+. Кросс-сборка под все три цели из любой ОС:

```sh
GOOS=windows GOARCH=amd64 go build -o dist/pashman-win.exe .
GOOS=darwin  GOARCH=arm64 go build -o dist/pashman-mac-arm64 .
GOOS=darwin  GOARCH=amd64 go build -o dist/pashman-mac-intel .
```

(На Windows PowerShell — задавать `$env:GOOS`/`$env:GOARCH` перед `go build`.)

## Структура

```
go.mod              модуль pashman
main.go             весь бэкенд (stdlib only)
web/index.html      весь интерфейс (vanilla JS), встраивается через //go:embed
dist/               бинарники + ИНСТРУКЦИЯ.md для пользователя
```

Папка `collections/` создаётся рядом с бинарником в рантайме и в репозиторий не входит
(может содержать личные данные и токены).
